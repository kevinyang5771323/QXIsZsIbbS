# Vue3宠物领养后台管理系统 java242

## 项目概述

本项目是一款基于Vue3的纯前端宠物领养后台管理系统。系统功能全面，界面美观，采用目前流行的前端技术栈开发，适用于学习Vue3 Composition API和后台管理系统实践。

## 技术栈

- **前端框架**：Vue3
- **组件库**：Element Plus
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **HTTP请求库**：Axios
- **图表库**：ECharts
- **日期处理库**：Day.js
- **工具库**：Lodash
- **CSS预处理**：Sass

## 功能模块

系统包含以下11个主要页面：

1. 仪表盘首页
2. 宠物分类管理
3. 宠物信息管理
4. 领养申请管理
5. 用户管理
6. 角色管理
7. 权限管理
8. 系统设置
9. 登录页
10. 注册页
11. 错误页面

### 自定义组件

- 包含15个以上自定义组件。

### 特性

- **分页功能**
- **状态管理**：使用Pinia实现状态管理
- **数据可视化**：ECharts图表展示
- **HTTP请求封装**：Axios拦截器统一处理
- **组件通信**：支持父子组件通信（props、emit）
- **指令支持**：支持常见指令如 `v-if`、`v-for`、`v-on`、`v-model`、`v-bind`、`v-permission`
- **计算属性与侦听器**：支持计算属性（computed）和侦听器（watch）
- **权限控制系统**：基于角色的权限管理

## 目录结构

```
src/
|-- api/               # API接口
|-- assets/            # 资源文件
|-- components/        # 自定义组件
|-- views/             # 页面文件
|-- router/            # 路由配置
|-- store/             # 状态管理
|-- utils/             # 工具方法
|-- App.vue            # 主应用组件
|-- main.js            # 入口文件
```

## 部署步骤

1. 克隆或下载项目代码。
2. 安装依赖：`npm install`。
3. 运行本地开发环境：`npm run dev`。
4. 构建生产环境代码：`npm run build`。
